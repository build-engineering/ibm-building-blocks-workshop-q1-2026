# Welcome to the the IBM Building Blocks Workshop - Q1 2026
In this workshop, you'll learn advanced techniques for using IBM Bob to ideate on partner use cases that incorporate the [IBM Building Blocks](https://ibm-self-serve-assets.github.io/building-blocks-docs/), build multi-agentic websites using those Building Blocks and then instrument them to observe their performance once deployed into production.

## Labs
Labs in this workshop are organized around the two Core Capability areas in the IBM Building Blocks:

They provide advanced training in these topics :
- Advanced agentic engineering using IBM Bob
- Multi-Agent Orchestration
- Agent Ops
- Data for AI
- Automation

All labs are built around how to best use IBM Bob to accelerare use of the IBM Building Blocks during the development of agentic applications and automation workflows. It's recommended to complete lab within each topic area in-order, but if you encounter this repository outside of a workshop, then jump into any lab that you find interesting.

### Lab 1. Advanced IBM Bob and the IBM Building Blocks (1 hour)
This lab teaches best practices for advanced AI engineering using IBM Bob as your AI coding assistant.  You'll learn what happens when IBM Bob is exposed to Captchas. You'll have fun building a creative web application while learning about topics like prompt caching plus visit your personal Bobalytics dashboard.

🤖 [Get started with Lab 1](lab-1-advanced-ibm-bob/README.md)

### Lab 2: Write a Custom Mode to build a data analytics website from wireframe images using IBM Carbon and a code execution MCP Server (1 hour)
Go deeper into Bob's mode and MCP server marketplace.  Transform wireframe images into an IBM Carbon-based REACT website.  Learn to execute arbitrary (but secure) code in an MCP Server. More important, build skills that will help you generate production-quality code faster.

🤖 [Get started with Lab 2](lab-2/README.md)

### Lab 3: Agent Building Blocks and Multi-Agent Orchestration (2 hours)

Context is critical, and when using AI coding assistants, the more **high quality** context the better.  Where high quality is the critical part when asking Bob to produce production quality code ready for deployment by IBM partners.   

🤖 [Get started with Lab 3](lab-3/README.md)

### Lab 4: Agent Ops (2 hours):

Evaluate, analyze, and security-test watsonx Orchestrate agents using the WXO ADK's build-time evaluation capabilities.

🤖 [Get started with Lab 4](lab-4/README.md)

### Lab 5: Data for AI Labs (1.5 hours)

Use Data for AI to enrich unstructured product content, store into a vector database then use to generate useful insights.

🤖 [Get started with Lab 5](lab-5/README.md)

### Lab 6: Automation Labs (5 hour)

Learn modern automation practices using Ansible and Terraform, with hands-on exercises deploying applications on OpenShift. Includes secrets management with HashiCorp Vault, application observability with IBM Instana, and automated resilience with IBM Concert.

🤖 [Get started with Lab 6](lab-6-automation/README.md)
